# FSF Tickets Stade

An international **stadium and event ticketing platform** built with Laravel — 
handling seat management, booking workflows, and payment processing.

## Features

- 🎫 Event and stadium ticket booking system
- 💺 Seat selection and availability management
- 💳 Payment gateway integration
- 📧 Booking confirmation and e-ticket generation
- 👥 Role-based access — Admin, Event Manager, Customer
- 📊 Sales reports and booking analytics

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Laravel, PHP |
| Frontend | Blade, Bootstrap, JavaScript |
| Database | MySQL |
| Deployment | cPanel / WHM |
